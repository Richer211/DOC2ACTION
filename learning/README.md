# Doc2Action 学习目录导航（分类版）

> 目标：让你按“学习 -> 复习 -> 面试”快速定位资料，不被文件数量干扰。

---

## A. 学习主线（先看这里）

- `学习主线/一键学习入口.md`：每天直接执行的学习步骤（推荐）
- `学习主线/学习导航总览.md`：从当前阶段到下阶段的复习路线与节奏
- `学习主线/学习开发双周期计划表.md`：4 周时间盒计划（学习优先）
- `学习主线/掌握标准与验收方式.md`：如何判断“真的掌握了”

建议顺序：

1. `学习主线/学习导航总览.md`
2. `学习主线/一键学习入口.md`
3. `学习主线/掌握标准与验收方式.md`
4. `学习主线/学习开发双周期计划表.md`

---

## 快速复习入口（30-60 分钟）

1. 看 `核心知识讲解/main.py初学者讲解.md`（10-15 分钟）
2. 答 `题库与面试/题库-main.py-面试与验收.md`（15-20 分钟）
3. 对照 `题库与面试/面试准备.md` 练 30 秒 + 2 分钟表达（10-15 分钟）
4. 在 `开发过程与计划/学习日志.md` 记录错题与下一步（5 分钟）
